# Actividad 1 - Ejercicios HTML, CSS y JavaScript

Este repositorio contiene 18 ejercicios de práctica donde se aplican conceptos básicos de desarrollo web.

---

## Descripción de actividades

### Ejercicio 1
Introducción a HTML básico.

### Ejercicio 2
Uso de encabezados, párrafos y listas.

### Ejercicio 3
Creación de enlaces entre páginas.

### Ejercicio 4
Uso de imágenes en HTML.

### Ejercicio 5
Tablas en HTML.

### Ejercicio 6
Formularios básicos.

### Ejercicio 7
Validación simple de formularios con JavaScript.

### Ejercicio 8
Uso de estilos con CSS.

### Ejercicio 9
Manipulación del DOM con JavaScript.

### Ejercicio 10
Eventos en botones.

### Ejercicio 11
Condiciones en JavaScript.

### Ejercicio 12
Ciclos en JavaScript.

### Ejercicio 13
Funciones básicas en JavaScript.

### Ejercicio 14
Interacción entre HTML, CSS y JS.

### Ejercicio 15
Proyecto final integrando lo aprendido.
